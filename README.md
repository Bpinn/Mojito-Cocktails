# 🍸 GSAP Cocktail Website

A modern and interactive cocktail website built with React, Tailwind CSS, and GSAP. This project focuses on creating a premium user experience with smooth animations, responsive layouts, and engaging UI interactions.

## 🤖 Introduction

This project is a visually rich cocktail landing page designed with modern frontend technologies. It includes GSAP-powered animations, interactive cocktail navigation, animated sections, and a responsive design that works smoothly across different screen sizes.

The main goal of this project was to practice building an animation-focused React application while maintaining clean component architecture, reusable code, and a polished user interface.

## ⚙️ Tech Stack

### React

React is used to build the application's component-based architecture. It helps create reusable components, manage UI state, and handle interactive features.

### GSAP

GSAP (GreenSock Animation Platform) is used to create smooth and performant animations, including:

- Element transitions
- Timeline animations
- Scroll-based effects
- Image and text animations

### Tailwind CSS

Tailwind CSS is used for building a responsive and modern interface with a utility-first styling approach.

### Vite

Vite provides a fast development environment with instant hot module replacement and optimized production builds.

## 🚀 Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have installed:

- Git
- Node.js
- npm

### Clone the Repository

```bash
https://github.com/Bpinn/Mojito-Cocktails.git
```

Navigate into the project:

```bash
cd your-repository-name
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

## 📦 Production Build

To create a production build:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
